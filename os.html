What is an Operating System?

→ An operating system (OS) is system software that acts as an intermediary between the hardware and software applications, allowing programs to run and providing a user interface. Essentially, the OS ensures that all parts of the computer work together efficiently.


FEATURES OF OPERATING SYSTEM

  1. PROCESS MANAGEMENT

    • Feature: The Os handles the creation, scheduling, and termination of processes.

    • Description:
      • Manages CPU aalocation among processes.

      • Supprots multitasking and multiprocessing.

      • Ensures proper synchronization between processes.

  2. MEMORY MANAGEMENT


    • Feature: Provides a structure for storing, retrieving, and organizing data.

    • Description:
      • Maintains file directories and access permissions.

      • Handles file creation, deletion, and access.

      • Supports file systems like FAT32, NTFS, ext4.


  3. FILE SYSTEM MANAGEMENT


    • Feature: Provides a structure for storing, retrieving, and organizing data.

    • Description:
      • Provides device drivers for hardware interaction.

      • Facilities communication between devices and applications.

      • Ensures smooth functioning of peripherals like printers, keyboards and hard drive.


  4. USER INTERFACE


    • Feature: Provides an interface for users to interact with the system.

    • Description:
      • Graphica User Interface (GUI): Point-and-click interface with windows, icons and users.

      • Command-Line Interface (CLI):Text-based interface for advanced users.

  5. NETWORKING


    • Feature: Facilities communication between systems over networks.

    • Description:
      • Supports internet and intranet connectivity.

      • Implements protocals like TCP/IP for data transfer.

      • Manages shared resources like files and printers.
